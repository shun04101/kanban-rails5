# kanban

## サイト概要
Trello風タスク管理アプリ

### サイトテーマ
カードと呼ばれる付箋みたいなものを動かしながらタスクを視覚的に管理できるサービス

### テーマを選んだ理由
Ruby on Railsの基礎学習としてCRUDアプリケーションの制作を行う

### 機能
- ユーザー登録
- リスト作成/編集/削除
